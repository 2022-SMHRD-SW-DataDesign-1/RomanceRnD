## 🌐 서비스 소개
* 서비스명: Mentorvation
* 서비스설명: 추천 시스템 기반 멘토 만남 서비스

<br>

## 📅 프로젝트 기간
2022.11.03 ~ 2022.11.11 (2주)
<br>

## 🧑‍🔧 주요 기능
* 사용자 기반 영상 추천
* 사용자 기반 멘토 프로필 추천
* 핸드쉐이크(팔로우 기능) 기능
* 티타임(멘토 예약) 기능
* 업로드 기능
* 썸네일 자동 생성 기능
* 스케쥴 예약
